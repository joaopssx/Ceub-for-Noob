A ideia e voce construir.

## Como executar

Quando criar `calculadora.py`:

Windows:

```bash
python calculadora.py
```

Linux/macOS:

```bash
python3 calculadora.py
```

## Saida esperada (exemplo)

```text
=== CALCULADORA ===
1) Somar
2) Subtrair
3) Multiplicar
4) Dividir
Escolha: 1
Digite dois numeros: 10 5
Resultado: 15.0
Deseja continuar? (s/n): n
```

## Erros comuns + como resolver

- divisao por zero sem tratamento
- loop que nunca termina
- entrada de texto quando espera numero

## Exercicios

### Faceis (3)

1. Implementar soma e subtracao.
2. Adicionar multiplicacao.
3. Adicionar divisao com validacao de zero.

### Medios (2)

1. Colocar menu numerico.
2. Permitir varias contas no mesmo programa.

### Desafio (1)

1. Separar cada operacao em funcao e manter `main` organizado.

## Requisitos minimos

- ler dois numeros
- ler operacao
- mostrar resultado
- tratar divisao por zero
- perguntar se quer continuar

## Extras opcionais

- historico de operacoes
- validacao forte de entrada
- modo "expressao direta" (ex: `2 + 3`)
