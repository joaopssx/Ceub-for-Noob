# Modulo Python - CEUB for Noob

Se C te da base de logica e memoria, Python te ajuda a ganhar velocidade.
Aqui a ideia e aprender sem pular etapa.

> Material independente, nao oficial.

## O que voce vai aprender aqui

- instalar Python e testar no terminal
- criar ambiente virtual (`venv`)
- usar `if`, `for`, `while` e funcoes
- trabalhar com listas, strings e dicionarios
- resolver exercicios e fechar com mini-projeto

## Ordem recomendada (nao pula)

1. Setup
2. Hello World
3. Condicionais
4. Repeticao
5. Funcoes
6. Listas e strings
7. Dicionarios
8. Lista de exercicios
9. Mini-projeto
