# Modulo Git + GitHub - CEUB for Noob

Esse modulo e pra te deixar seguro com versionamento.
Depois que voce pega isso aqui, para de perder arquivo e comeca a trabalhar em equipe direito.

> Material independente, nao oficial.

## O que voce vai aprender aqui

- instalar e configurar Git
- criar repositorio local
- fazer commit sem bagunca
- usar branch e merge
- subir projeto pro GitHub
- abrir Pull Request
- resolver conflito sem panico

## Ordem recomendada (nao pula)

1. Setup
2. Init / add / commit
3. Branch e merge
4. Remote no GitHub
5. Colaboracao com PR
6. Conflitos
7. Exercicios
8. Mini-projeto de workflow

## Regras do jogo

- commit pequeno e com mensagem clara
- branch por tarefa
- nao manda codigo sem testar

> Git e habito. Quanto mais voce usa, mais natural fica.

## Padrao dos exemplos do modulo

```bash
git status
git add .
git commit -m "feat: adiciona exemplo"
```
