# Mini-projeto 01 - Boletim no terminal (Portugol)

## Objetivo

Criar um mini sistema de boletim para praticar toda base do modulo.

## Explicacao curta

Projeto simples, mas muito bom pra consolidar logica:

- entrada de dados
- condicional
- repeticao
- vetor
- funcao

## Codigo completo

Nao tem codigo pronto aqui.
A proposta e voce montar.

## Como executar

1. Abra Portugol Studio.
2. Crie programa novo.
3. Implemente as features.
4. Execute com `F6`.
5. Teste com diferentes notas.

## Saida esperada (exemplo)

```text
=== BOLETIM ===
Nome: Ana
Nota 1: 7
Nota 2: 8
Nota 3: 6
Media: 7
Situacao: Aprovado
```

## Erros comuns + como resolver

- media calculada errada por tipo inteiro
- condicao de status invertida
- loop com limite errado ao ler notas

## Exercicios

### Faceis (3)

1. Ler nome e 3 notas.
2. Calcular media.
3. Mostrar status aprovado/reprovado.

### Medios (2)

1. Permitir cadastrar 3 alunos.
2. Mostrar maior media da turma.

### Desafio (1)

1. Criar menu: cadastrar, listar, buscar aluno por nome.

## Requisitos minimos

- cadastro de aluno
- leitura de 3 notas
- calculo de media
- status final

## Extras opcionais

- registro para aluno
- vetor de alunos
- funcao para calcular media
- ranking das medias
