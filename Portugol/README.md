# Modulo Portugol - CEUB for Noob

Se voce esta comecando do zero, Portugol e uma otima ponte pra logica.
A ideia aqui e aprender raciocinio sem se preocupar com detalhe de linguagem mais avancada.

> Material independente, nao oficial.

## O que voce vai aprender aqui

- instalar e abrir Portugol Studio
- criar seu primeiro algoritmo
- entrada e saida com `leia` e `escreva`
- condicional com `se/senao`
- repeticao com `enquanto` e `para`
- funcoes/procedimentos
- vetores, matrizes e registro

## Regras do jogo

- rode o algoritmo e teste entradas diferentes
- se der erro, leia a linha que o Portugol Studio marcou
- mexa no exemplo e rode de novo

> Se voce so ler e nao executar, parece facil. Quando roda, aprende de verdade.

## Padrao dos exemplos

```portugol
programa
{
    funcao inicio()
    {
        escreva("Exemplo em Portugol")
    }
}
```
