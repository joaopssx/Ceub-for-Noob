# 01 - Primeiro programa em Portugol

## Objetivo

Criar e executar seu primeiro algoritmo com mensagem no console.

## Explicacao curta

Esse e o famoso Hello World.
Serve pra validar ambiente e entender estrutura minima.

## Codigo completo

```portugol
programa
{
    funcao inicio()
    {
        escreva("Hello, Portugol!")
    }
}
```

## Como executar

1. Abra Portugol Studio.
2. Clique em `Novo programa`.
3. Cole o codigo.
4. Clique em `Executar` ou `F6`.

## Saida esperada (exemplo)

```text
Hello, Portugol!
```

## Erros comuns + como resolver

- esqueceu `funcao inicio()`
- faltou fechar `}`
- texto com aspas erradas

## Exercicios

### Faceis (3)

1. Mostrar seu nome.
2. Mostrar seu curso.
3. Mostrar duas frases em sequencia.

### Medios (2)

1. Montar mini cabecalho com 3 linhas.
2. Usar `escreva("\n")` para pular linha.

### Desafio (1)

1. Criar um menu textual com 3 opcoes.
