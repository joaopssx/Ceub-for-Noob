# 00 - Instalacao do Portugol Studio

## Objetivo

Instalar Portugol Studio e validar que voce consegue rodar um algoritmo.

## Explicacao curta

Portugol Studio ja vem pronto pra estudar logica.
Voce instala, abre e executa. Bem mais rapido pra comecar.

## Codigo completo

Nao tem codigo aqui. Esse capitulo e de ambiente.

## Como executar

### Windows / Linux / macOS

1. Baixe em: https://portugol.dev/
2. Instale normalmente.
3. Abra o Portugol Studio.
4. Clique em `Novo programa`.
5. Rode qualquer exemplo com `F6`.

Se abrir e executar, ta pronto.

## Saida esperada (exemplo)

```text
Algoritmo executado com sucesso.
```

## Erros comuns + como resolver

### programa nao abre

- reinstale com a versao mais recente
- verifique permissao de execucao no Linux/macOS

### erro ao executar

- veja se o codigo tem `funcao inicio()`
- confira chaves `{}` abertas/fechadas

### interface em branco ou travando

- feche e abra novamente
- atualize driver de video (em alguns casos)
