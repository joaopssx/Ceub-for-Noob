# Modulo C - CEUB for Noob

Se voce quer montar base de programacao de verdade, comeca por aqui.
Esse modulo foi feito pra noob: simples, na ordem certa e com bastante pratica.

> Material independente, nao oficial.

## O que voce vai aprender aqui

- Preparar ambiente sem dor de cabeca.
- Compilar e rodar no terminal.
- Tomar decisao com `if/else`.
- Repetir com `for`.
- Organizar codigo com funcoes.
- Trabalhar com vetores e iniciar ponteiros.

## Ordem recomendada (nao pula)

1. Setup
2. Hello World
3. if/else
4. for
5. Funcoes
6. Vetores
7. Ponteiros
8. Lista de exercicios
9. Mini-projeto
